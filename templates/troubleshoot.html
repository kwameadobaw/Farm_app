PDF Generation Issues

Common Error: "Error generating PDF"

This error occurs when the application encounters issues with WeasyPrint, which is used for generating PDF files. Follow these steps to resolve the issue:

Step 1: Verify WeasyPrint Installation

First, check if WeasyPrint is properly installed in your Python environment:

  • Open a command prompt or terminal
  • Run: pip list | findstr WeasyPrint (Windows) or pip list | grep WeasyPrint (Linux/macOS)
  • If not found or showing an older version, reinstall it with: pip install WeasyPrint==59.0

Step 2: Install Required Dependencies

WeasyPrint requires some system libraries to function properly:

  • Windows: Install Microsoft Visual C++ Redistributable (both x86 and x64 versions)
  • Linux: Install required libraries with:
    apt-get install build-essential python3-dev python3-pip python3-setuptools python3-wheel python3-cffi libcairo2 libpango-1.0-0 libpangocairo-1.0-0 libgdk-pixbuf2.0-0 libffi-dev shared-mime-info
  • macOS: Install dependencies with:
    brew install cairo pango gdk-pixbuf libffi

Step 3: Restart the Application

After making any changes:

  • Restart the Flask application
  • Try generating a PDF again

Note: WeasyPrint is a pure Python library that converts HTML/CSS to PDF without requiring external executables. If you're still experiencing issues, check the official WeasyPrint documentation for troubleshooting.

Admin Access Issues

If you're having trouble accessing the admin page:

Default Login Credentials:

  • Username: admin
  • Password: farmapp123

If you need to reset the admin password:

  1. Open app.py in a text editor
  2. Locate the ADMIN_USERNAME and ADMIN_PASSWORD variables near the top
  3. Change the values as needed
  4. Save the file and restart the application
